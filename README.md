# MotionCut Project 2 - E-Commerce Product Page UI

A modern and conversion-focused mobile product detail page UI for an e-commerce application.

## 📱 Screen Preview (Video)
<video src="project-2-video.mp4" width="350" controls></video>

## 🎯 Features Designed
- **Header:** Back button, wishlist icon, and cart icon.
- **Product Showcase:** Large image section with carousel indicators.
- **Product Details:** Price details ($179.00), ratings, and description.
- **Selectors:** Size (S, M, L) and color selection chips along with a quantity selector.
- **Sticky Bottom Area:** Clean "Add to Cart" and "Buy Now" buttons.
